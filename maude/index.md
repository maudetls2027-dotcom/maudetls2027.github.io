# Formal Maude Model

This directory contains the executable Maude model used as the
specification-level oracle for TLS handshake testing.

The model includes RFC-derived requirements, TLS message and cryptographic
abstractions, API-level specifications, and scenario definitions used to drive
deviation-guided test generation.
